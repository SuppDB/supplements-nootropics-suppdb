# Assets

Put marketing images here and reference them from the top-level `README.md`, e.g.:

```markdown
![SuppDB data preview](assets/preview.png)
```

Suggested assets to add:
- `kaggle-cover.png` — hero/cover banner used in the README and on Kaggle.
- `preview.png` — a screenshot of the dataset (a few rows in a spreadsheet or DB browser).
- `landing.png` — a screenshot of the suppdb.net landing page.

(GitHub renders the top `README.md`, so a hero image here makes the repo landing far more compelling.)
